## 1. O que você está montando (visão geral)

Imagine o Instagram como uma **vitrine**:

| Parte do perfil | O que é (em português claro) | Para que serve |
|-----------------|------------------------------|----------------|
| **Foto + bio + link** | Cartão de visita no topo | Dizer o que você faz e para onde mandar clicar |
| **Feed** | Posts que ficam no perfil para sempre | Apresentação, educação, prova de trabalho |
| **Stories** | Vídeos/fotos verticais que somem em 24h | Conteúdo rápido, enquetes, bastidores |
| **Destaques** | Pastas fixas embaixo da bio | Catálogo permanente por tipo de negócio (Saúde, Varejo…) |
| **Reels** | Vídeos curtos (como TikTok) | Mostrar o site rolando no celular — isso converte muito |

**Meta comercial:** alguém vê um modelo de site → clica no WhatsApp → pede orçamento.

## 2. Glossário rápido

| Termo | Significa |
|-------|-----------|
| **Bio** | Texto curto abaixo do nome + **um link** (só cabe 1; use o portfólio) |

## 4. Anatomia de um destaque (a parte mais importante)

Cada destaque (Saúde, Varejo, Orçamento…) segue **sempre o mesmo roteiro de 5 stories**:

```
Story 1  →  Gancho (imagem feita no MyHub)
             "Site para Saúde — a partir de R$ 300"

Story 2  →  Demo 1 (GRAVAR tela do celular)
             Abrir landing-nutricionista no Chrome, scroll até o WhatsApp

Story 3  →  Demo 2 (GRAVAR)
             Outro site do mesmo segmento

Story 4  →  Demo 3 ou detalhe (GRAVAR)
             Formulário, animação, painel admin…

Story 5  →  CTA (imagem no MyHub)
             "Quero um assim" + sticker de link pro WhatsApp
```

Depois de publicar os 5 stories **no mesmo dia**:

1. Toque no seu story (canto inferior direito do app).
2. Selecione os 5.
3. Toque **Destacar** → escolha o destaque (ex.: Saúde) ou crie um novo.
4. A **capa redonda** você gera no MyHub (prompts no arquivo de prompts).

## 7. Como gravar um Reel ou story de demo (passo a passo)

### No celular (mais fácil)

1. Abra o Chrome no celular.
2. Acesse a URL da demo (ex.: `https://tofariasti.github.io/landing-nutricionista/`).
3. Ative **gravação de tela** (Android: painel rápido; iPhone: Central de Controle).
4. Role a página **devagar**: hero → serviços → botão **WhatsApp**.
5. Pare a gravação (15–30 segundos é ideal).
6. No Instagram: **+** → **Reel** → importe o vídeo → legenda pronta no editorial.

### No computador (DevTools)

1. Chrome → F12 → ícone de celular → largura **390px**.
2. Abra a demo.
3. Grave com OBS, QuickTime ou extensão de screen record.
4. Envie pro celular ou publique pelo Meta Business Suite.

**O que sempre mostrar:** scroll + botão verde do WhatsApp.

## 8. Como gerar imagens no MyHub IA

Site: [myhub.ia.br/images](https://myhub.ia.br/images)

### Fluxo básico (repita para cada imagem)

1. Escolha a **proporção**:
   - **1:1** → capas de destaque, carrosséis do feed
   - **9:16** → stories (gancho e CTA)
2. Abra [`instagram-prompts.md`](instagram-prompts.md).
3. Copie o **bloco de estilo** (Neon ou Flat) + o prompt do slide.
4. Cole no campo "Descreva a imagem…".
5. Gere **uma imagem por vez**.
6. Se o texto sair errado em português → regenere **sem texto** e coloque o texto no **Canva**.

### Qual estilo usar?

| Estilo | Use em |
|--------|--------|
| **Editorial Flat** | Capas redondas dos destaques, stories, carrosséis educativos |
| **Neon Premium** | Lançamento do perfil, apresentação, cases (TechDrone360) |

### O que NÃO gerar no MyHub

- Stories 2, 3 e 4 dos destaques (são gravações de site).
- Reels (só a **capa/thumbnail** opcional no MyHub).

## 10. Publicar um carrossel (5 slides)

1. Gere os 5 slides no MyHub (1:1, um por vez; anexe o slide anterior para manter o estilo).
2. Instagram → **+** → **Post** → selecione as 5 imagens **na ordem**.
3. Cole a legenda do editorial §5.
4. Publique.
5. Se o post alimenta um destaque, grave stories extras no mesmo dia e arquive no destaque certo.

---

## 11. Publicar um Reel

1. Grave a demo (§7).
2. (Opcional) Gere thumbnail 9:16 no MyHub.
3. Instagram → **+** → **Reel** → importe o vídeo.
4. Legenda modelo (trocar `[SEGMENTO]`):

   ```
   Site para Nutricionista — modelo pronto para personalizar ✨

   ✅ Layout profissional no celular
   ✅ Botão WhatsApp integrado
   ✅ A partir de R$ 300

   Quer um assim? Link na bio ou chama no direct.
   .
   .
   #landingpage #siteprofissional #nutricionista #fariasdigital #portoalegre
   ```

5. No mesmo dia: publique stories do segmento e **arquive no destaque** correspondente.

## 12. Mapa mental — o que fazer hoje

Se você só puder fazer **uma coisa hoje**, siga esta ordem:

```
Dia 1   Bio + link + foto de perfil
Dia 2   Gerar 3 capas: Orçamento, Saúde, Como funciona (MyHub 1:1)
Dia 3   Gravar 3 demos de Saúde + gerar stories 1 e 5 (MyHub 9:16)
Dia 4   Publicar stories → criar destaque Saúde
Dia 5   Carrossel de apresentação (5 slides) + Reel Nutricionista
Dia 6   Repetir lógica para destaque Orçamento
Dia 7   Destaque Como funciona (4 passos do processo, sem site demo)
```

Depois disso, siga o calendário semanal do editorial.

## 13. Erros comuns (evite)

| Erro | O certo |
|------|---------|
| Gerar vídeo de site no MyHub | Gravar tela real |
| Destaque com 1 story só | Esperar os 5 stories do roteiro |
| Texto ilegível da IA | Refazer no Canva com Plus Jakarta Sans |
| Esquecer o WhatsApp no Reel | Sempre terminar o scroll no botão verde |
| Publicar story e não arquivar | Arquivar no destaque **no mesmo dia** |
| Capa destaque com muito texto | Ícone simples no centro (crop circular corta as bordas) |
